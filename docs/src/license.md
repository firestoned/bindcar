## Third-Party Licenses

bindcar includes and depends on various open-source libraries. Here are the key dependencies:

### Rust Dependencies

- **axum** - Web framework (MIT)
- **tokio** - Async runtime (MIT)
- **serde** - Serialization framework (MIT OR Apache-2.0)
- **tracing** - Logging and diagnostics (MIT)
- **tower** - Service middleware (MIT)

For a complete list of dependencies and their licenses, see the `Cargo.toml` file and run:

```bash
cargo tree
```
